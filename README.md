# SnapScroll

个人用跨平台截图工具：区域截图、全屏截图、**任意窗口滚动长截图**、贴图置顶。

## 快捷键

| 功能 | 快捷键 |
|------|--------|
| 截图（框选后选「区域」或「长截图」） | `F1` |
| 全屏截图 | `F2` |

长截图进行中再按一次 `F1` 亦可完成拼接。

任务栏右下角托盘图标右键菜单（中文）：截图 / 全屏截图 / 显示主窗口 / 打开保存目录 / 退出。  
关闭主窗口会隐藏到托盘，不会退出。

截图后自动复制到剪贴板（不弹贴图窗口），并静默保存到 `E:\截图文件`（没有则自动创建）。

## 启动 / 开机自启

- 日常启动：双击 `启动截图助手.vbs`（无黑窗口）或 `启动截图助手.bat`
- 开机启动：已写入 Windows「启动」文件夹快捷方式「截图助手」
  - 路径：`%APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup\截图助手.lnk`
  - 开机以 `--silent` 静默运行：只挂托盘，不弹主窗口；需要时右键托盘「显示主窗口」

## 滚动长截图用法

1. `F1` 框选后点「长截图」图标（第一次 = 开始）
2. 选区层收起后，用鼠标滚轮滚动桌面上你要截的窗口（滚到哪算哪）
3. 停稳会自动截取新帧；滚够后点浮条上的**长截图完成图标**，或再按一次 `F1`（第二次 = 完成）
4. `Esc` 取消

提示：开始后不抢鼠标焦点，可直接滚当前鼠标下的窗口。

## 开发 / 运行

需要 Node.js 18+。大陆网络建议先开代理：

```powershell
$env:HTTP_PROXY="http://127.0.0.1:7890"
$env:HTTPS_PROXY="http://127.0.0.1:7890"
npm install
npm run dev
```

打包：

```powershell
npm run dist
```

## 技术说明

- Electron + TypeScript：跨平台桌面壳
- `screenshot-desktop`：屏幕捕获
- `@nut-tree-fork/nut-js`：模拟点击与滚轮（任意窗口）
- `jimp`：区域裁剪与重叠区拼接

首次在 Windows 上做滚动截图时，系统可能弹出辅助功能 / 输入模拟相关权限提示，请允许。
